# Assets Directory

Place your image and video assets here:

## Required Files:

1. **incub-logo.png** - Main Invertis Incubation logo (transparent background recommended)
   - Recommended size: 400x400px
   - Format: PNG with transparency

2. **main logo.png** - "Powered By" logo
   - Recommended size: 200x100px
   - Format: PNG

3. **Incvid.mp4** (Optional) - Hero section background video
   - Recommended: 1920x1080 (Full HD)
   - Format: MP4 (H.264 codec)
   - Duration: 10-30 seconds loop
   - Keep file size < 10MB for faster loading

## Copy from Root Project:

You can copy these assets from your root project directory:
- `/assets/incub-logo.png` → here
- `/assets/main logo.png` → here
- `/assets/Incvid.mp4` → here

## Other Assets to Consider:

- Team member photos
- Partner logos
- Event photos
- Success story images
- Testimonial images

Once copied, restart your development server for the images to appear.
